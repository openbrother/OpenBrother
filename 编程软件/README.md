1．下载完成后解压，直接按照指引进行安装

2．安装完成后双击桌面上的“PZStudio”![img](file:///C:\Users\向金\AppData\Local\Temp\ksohtml13820\wps1.jpg)，如果您是第一次打开，系统会自动提示您进行升级，请保持网络畅通，并且点击升级按钮，等待系统自动升级完成。
